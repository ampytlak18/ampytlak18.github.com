# bulldogearthday.github.com
Main 2D site for Fresno State Earth Day

You’ll find the blog posts in the `_posts` directory.

To add new posts, simply add a file in the `_posts` directory that follows the convention `YYYY-MM-DD-name-of-post.ext` and includes the necessary front matter. Take a look at the source for the welcome post to get an idea about how it works.
